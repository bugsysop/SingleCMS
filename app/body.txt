<h2>Welcome to Single CMS</h2>
<p>Single CMS is a simple, safe, editor for a single web page.</p>
<p>It allows a web designer to build a single page site, while allowing a client full control over the content</p>
<h2>Features</h2>
<ul>
<li><strong>Simple</strong> Easy to use interface</li>
<li><strong>Secure</strong> Control which tags are permitted in page content</li>
<li><strong>Lightweight</strong> Requires no database back end</li>
</ul>
<h2>Getting Started</h2>
<p>Using SingleCMS is easy, just follow these steps.</p>
<ol>
<li>Unpack the zip file on to your PHP server</li>
<li>Make <code>body.txt</code> writeable by your web server</li>
<li>Define the basic settings in <code>config.php</code></li>
<li>That's it!</li>
</ol>
